1

STEP 01·ORIENT

Set up your warehouse structure

Warehouse Management is the physical layer — warehouses, then Locations and UoM Groups under Setup. Head Office already has its Default Warehouse, marked Active; add more to model extra sites.

2

STEP 02·CLICK

Receive goods with a procurement order

Under Procurement, click New Procurement Order to bring goods in from a supplier. Receiving against it posts the units into stock at the head-office warehouse.

3

STEP 03·CLICK

Auto-replenish from reorder rules

Replenishment turns min-max rules into purchase orders. Use Dry Run to preview, then Run & Generate POs to raise them automatically — or New Rule to define a fresh reorder point.

4

STEP 04·CLICK

Batch picks into a wave

Pick Waves group outbound move-lines into one picking run — planned, released, in progress, completed. Click New Wave and select the move lines from the inventory queue to release them together.

5

STEP 05·CLICK

Count stock with an audit session

Stock Audit verifies physical inventory. Start a Spot Check for a few SKUs, a Cycle Count for a rolling subset, or a Full Count for everything — posted variances adjust on-hand automatically.

6

STEP 06·SAVE

Publish a standard cost

Standard Costs set the valuation baseline. Publish Cost fixes a SKU's standard; Compute Variance compares it to actual purchase price so purchase-price variance flows to the ledger.
